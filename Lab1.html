LAB1 PAGE


The intent of this lab is to start building a very simple, local drive based, website.
In this lab we have to create static web pages.
We have to create 3 pages:
1) Home page : It displays the name, course and the date.
Along with this it also shows the links to 2 different pages named
Lab1 Page and Sitemap Page.
2) Lab1 page : Lab1 Page provides the description of the lab and links
to HomePage and Sitemap Page.
3) Sitemap page: SiteMap Page provides the links to HomePage and Lab1 Page.
